== Description ==
NabigatuEuskaraz.com proiektuaren partaide izateko inolako koderik aldatu gabe. Plugin honi esker nahikoa da fitxategiak igotzea ta plugin-a gaitzea euskaraz mintzo ez diren nabigatzaileak ohartarazteko.

== Installation ==
1. Deskonprimitu fitxategia
2. Igo nabigatu-euskaraz direktorioa `/wp-content/plugins/` direktoriora
3. Gaitu nabigatu-euskaraz plugin-a WordPress-en 'Plugins' menutik
